    Ackerman(m, n) = Ackerman(m - 1, 1) si m > 0 y n = 0 \
    Ackerman(m, n) = Ackerman(m - 1, Ackerman(m, n - 1)) si m > 0 y n > 0 \
    Ej. Si se tiene Ackermann(1, 2) = 4; Ackermann(3, 2) = 29 \
